# HR-Analytics-Dashboard
🚀 HR Analytics Dashboard | Excel Project 📊
# 📌 Project Overview

This project is an HR Analytics Dashboard built using Microsoft Excel, created as part of my learning journey in data analytics.
The objective of this project is to analyze HR data and convert raw employee information into meaningful insights that can support HR and business decision-making.

# 🎯 Project Objectives

Analyze employee attrition and workforce trends

Identify key factors contributing to employee turnover

Create an interactive and visually intuitive HR dashboard

Practice real-world HR analytics using Excel

# 🔍 Dashboard Insights

The dashboard provides analysis on:

Total Employees, Active Employees & Attrition Rate

Job Satisfaction Rating

Average Employee Age

# 📊 Attrition Analysis by:

Gender

Department (HR, R&D, Sales)

Job Role

Education Level

Age Group

Marital Status

# 🛠 Tools & Techniques Used

Microsoft Excel

Data Cleaning & Data Structuring

Pivot Tables & Pivot Charts

Excel Formulas & Calculations

Conditional Formatting

Slicers for Interactive Filtering

Dashboard Design & Data Visualization

# 📈 Key Learnings

Understanding and calculating core HR metrics such as attrition rate and job satisfaction

Identifying high attrition risk groups based on age, education, and job roles

Improving data storytelling skills through effective dashboard design

Gaining hands-on experience in Excel-based analytics for HR use cases
